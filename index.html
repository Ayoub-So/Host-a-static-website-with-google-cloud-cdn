About This Project

This website is built using modern web technologies and deployed using Infrastructure as Code (Terraform). The architecture leverages Google Cloud Platform's robust services:

Google Cloud Storage Cloud CDN Terraform HTML5 CSS3

The site is stored in a Google Cloud Storage bucket and served through Cloud CDN for optimal performance and reliability.
