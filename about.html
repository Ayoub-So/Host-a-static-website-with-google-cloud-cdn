Project Overview

This project demonstrates how to host a static website efficiently on Google Cloud Platform (GCP) using Infrastructure as Code with Terraform. The website is stored in Google Cloud Storage and distributed globally through Google Cloud CDN.

This approach provides:

  • High availability and reliability
  • Low latency for users worldwide
  • Cost efficiency
  • Easy deployment and management
  • Built-in security features

Architecture 🏗️

The infrastructure consists of the following components:

Cloud Storage

Stores static files (HTML, CSS, JS, images)

Cloud CDN

Caches content globally for fast delivery

Load Balancer

Routes traffic to appropriate resources

SSL/TLS

Encrypts traffic with HTTPS

Technology Stack 🛠️

Infrastructure as Code

Terraform - We use Terraform to define and manage our infrastructure. This allows us to:

  • Version control our infrastructure
  • Replicate environments easily
  • Maintain consistency across deployments
  • Collaborate with team members

Cloud Platform

  • Google Cloud Storage - Object storage for static files
  • Google Cloud CDN - Content delivery network
  • Google Cloud Load Balancer - Traffic distribution
  • Cloud DNS - Domain name resolution

Frontend Technologies

  • HTML5 - Semantic markup
  • CSS3 - Modern styling and animations
  • JavaScript - Interactive features

Key Benefits 🎯

Performance

Google Cloud CDN caches your content at edge locations worldwide, reducing latency and improving page load times for users globally.

Scalability

Google Cloud Storage automatically scales to handle traffic spikes without any intervention from your side.

Security

Built-in security features include SSL/TLS encryption, DDoS protection, and fine-grained access controls.

Cost Efficiency

Pay only for the storage and bandwidth you use. No need to maintain expensive servers.

Reliability

Google Cloud's infrastructure provides 99.95% uptime SLA, ensuring your website is always available.

Easy Management

Use Terraform to manage your entire infrastructure as code, making updates and rollbacks simple.

Getting Started 🚀

To deploy this website:

  1. Configure your GCP project ID in terraform.tfvars
  2. Authenticate with Google Cloud using gcloud auth application-default login
  3. Run terraform init to initialize Terraform
  4. Run terraform plan to preview changes
  5. Run terraform apply to deploy the infrastructure
  6. Upload the HTML files to your Storage bucket
  7. Configure your domain and SSL certificate
